# Basic Maths Problems

This folder contains solutions to basic mathematics problems from Striver's A2Z DSA Sheet.

## Topics Covered

* Count Digits in a Number
* Reverse a Number
* Palindrome Number
* GCD (Greatest Common Divisor)
* Armstrong Number
* Print All Divisors
* Prime Number Check

## Concepts Used

* Modulo Operator (`%`)
* Integer Division
* Loops
* Number Manipulation
* Mathematical Properties
* Time Complexity Optimization

## Language

C++
